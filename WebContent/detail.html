로봇청소기의 기능

청소기 사용 고객은 웹 또는 App을 통해 각 방을 추가 또는 삭제하거나, 방의 특성 및 크기를 설정할 수 있습니다. 또 웹 또는 App을 통해 제품의 사진, 기능, 제작자등을 확인 할 수 있습니다. 로봇청소기의 동작은 등록된 방의 크기와 상태를 기반으로 작동합니다.

웹 기능

웹을 통해 방을 등록 또는 삭제하거나 방의 특성 및 크기를 변경할 수 있으며, '청소하기'메뉴를 통하여 (동작)또는 (예약)기능을 수행할 수 있습니다. (청소내역보기)를 통해 해당 방의 청소내역 또한 조회할 수 있습니다.

App 기능

App을 통해 방을 등록 또는 삭제하거나 방의 특성 및 크기를 변경할 수 있으며, '청소하기'메뉴를 통하여 (동작)또는 (예약)기능을 수행할 수 있습니다. (청소내역보기)를 통해 해당 방의 청소내역 또한 조회할 수 있습니다.

동작

사용자가 청소를 진행하길 원하는 방의 (동작) 버튼을 선택하면 로봇청소기에 동작실행이 전달되며 선택한 방의 청소가 시작 됩니다.

예약

사용자가 청소를 진행하길 원하는 방의 (예약) 버튼을 선택하면 로봇청소기에 예약시간을 설정하여 예약 시간 이후에 선택한 방의 청소가 시작 됩니다.

청소내역보기

사용자가 청소하였던 방의 (청소내역보기) 버튼을 선택하면 현재시간 기준으로 로봇청소기가 선택한 방의 청소시작시간과 끝난시간을 알 수 있습니다. 청소를 진행하지 않은 방의 경우 청소하지 않았음을 알 수 있습니다.

경쟁 제품들보다 우수한 기능

단 한 번 만에 전달

신뢰성의 핵심인 "단 한 번 만에 전달" 기능을 적용하여 청소기에 명령을 내리며 명령 데이터를 DB에 'CRUD'합니다.

DB를 활용한 강력한 정보 보호

예외상황으로 애플리케이션이 중단되어도 명령은 영구저장소 DB에 저장되어 있어 명령의 유실이 없으며, DB처리를 백엔드 내부에서 진행하여 개인 정보 보호 기능이 뛰어납니다.

웹과 APP을 통한 동시 접근 가능

접근성을 높이기 위하여 다양한 프론트엔드를 지원합니다. 웹과 앱을 동시 지원하여 어느 것을 사용하여도 동일한 결과를 얻을 수 있습니다.

웹페이지 소개

HOME

메인 홈페이지 화면 입니다.

해당 페이지에서 로봇청소기의 전체적인 사진과 함께 청소기의 간략한 기능을 나타냅니다. 또 페이지 하단부에 사용자가 설정한 방의 사진과 함께 방이름 및 간략한 정보를 표시해줍니다.

제품설명서

로봇청소기의 제작과정 및 결과물에 대한 자세한 정보를 나타내는 화면 입니다.

해당 페이지에서 로봇청소기의 사진과 함께 청소기의 기능을 알 수 있습니다. 로봇청소기의 조작 방법 및 웹페이지의 기능, app의 기능 및 사용자가 놓치기 쉬운 기능들과 제품의 자세한 정보에 대한 설명이 나타납니다.

청소시작

로봇청소기의 실행과 관련된 화면입니다.

해당 페이지에서 로봇청소기의 작동여부 및 예약 여부, 방의 등록 ,삭제 및 수정을 담당합니다. 등록의 경우 우측 상단의 (등록)버튼을 눌러 방을 등록할 수 있습니다. 방의 수정 또는 삭제의 경우 수정하고자 하는 방을 더블클릭하면 상세정보창이 생성됩니다. (동작)버튼을 통해 로봇청소기를 바로 실행 시킬수 있으며, (예약)버튼을 통해 로봇청소기를 현재시간 기준으로 몇분 뒤에 실행 할 지 정할 수 있습니다. (청소내역보기)를 통해 해당 방의 청소시간 및 청소내역리스트를 확인 할 수 있습니다.

제작자

로봇청소기의 제작자와 관련된 화면입니다.

로봇 청소기의 제작자 및 팀의 전반적인 프로필에 대하여 나타내주며, 각 인원이 맡은 주요 파트와 함께 문의사항을 문의할 수 있는 개별 연락처가 기록되어 있습니다.

웹 기능 사용방법








방 상세 정보 보기

해당하는 방의 상세정보를 보는 기능입니다.

방의 상세 정보를 보기 위해 테이블에서 해당 방의 더블클릭 이벤트를 통해 상세정보를 조회 할 수 있습니다. 상세보기를 통해 조회된 방은 설정된 가로길이와 세로길이를 표시해줍니다. 이 데이터를 수정하고 싶을 경우 해당 데이터를 재입력한 후 수정버튼을 누르면 됩니다. 해당 방의 상태정보를 삭제하고 싶으면 상세정보창의 하단부의 삭제하기 버튼을 누르면 해당방의 정보가 삭제됩니다.










방 청소 진행하기

해당하는 방의 청소를 진행하는 기능입니다.

확인을 선택할 경우 선택된 방의 청소를 진행합니다. 해당 버튼을 클릭하게되면 로봇청소기에 청소 진행 명령을 내림으로써 청소를 예약없이 즉시 진행하게 됩니다.









방 청조 예약 진행하기

해당하는 방의 청소를 예약을 진행하는 기능입니다.

해당 페이지를 활성화 할 경우 선택된 방의 청소 예약을 진행합니다. 예약 설정 시간은 선택가능하며, 선택된 시간은 현재시간 이후로부터 적용됩니다. 선택된 시간은 DB에 저장되며 저장된 DB의 데이터를 기반으로 로봇청소기에 자동으로 명령을 내려 설정된 예약시간에 정확하게 청소를 시작합니다.













방 청소 내역 조회하기

해당하는 방의 청소 내역을 조회하는 기능입니다.

해당 페이지를 활성화 할 경우 선택된 방의 청소 내역 조회를 진행합니다. 해당 방의 청소내역이 없을 경우 청소내역이 없음이 나타나며, 청소내역이 있을 경우 청소시작시간과 청소종료시간이 모두 표시되어 해당 페이지에서 확인 할 수 있습니다.














방 추가 등록 진행하기

방의 정보를 추가하는 기능입니다.

해당 페이지를 활성화 할 경우 새로운 방의 정보를 등록할 수 있습니다. ID의 경우 시스템 내부에서 자동적으로 번호를 부여하며, 방의 사용자 정보 및 방의 가로,세로 사이즈를 입력할 수 있습니다. 입력한 데이터는 DB에 저장되며 로봇청소기에 명령을 내릴 때 사용되는 데이터 입니다.
